# Cric Tracker Downloads

## Windows
Use `index.exe` if you want a Windows desktop launcher.

## Mac, Android, iPhone, iPad
Use `crictracker-pwa-for-mac-android-iphone.zip`.

Important: phones should not open the raw HTML file from storage. Upload the PWA folder to GitHub Pages, Netlify, or Vercel, then share the website link.

Install steps:
- Android: open the website link in Chrome > three dots > Add to Home screen / Install app.
- iPhone/iPad: open the website link in Safari > Share > Add to Home Screen.
- Mac: open the website link in Safari/Chrome > Add to Dock / Install app.

Each person gets their own private data saved on their own device/browser.
